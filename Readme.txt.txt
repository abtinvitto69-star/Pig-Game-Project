بازی Pig – بازی تاس با جاوااسکریپت

بازی Pig یک بازی دو نفره مبتنی بر تاس است که با استفاده از HTML، CSS و JavaScript توسعه داده شده است. بازیکنان به نوبت تاس می‌اندازند و امتیاز جمع‌آوری می‌کنند. اگر عددی به‌جز ۱ بیاید، امتیاز آن به امتیاز فعلی بازیکن اضافه می‌شود؛ اما اگر عدد ۱ ظاهر شود، امتیاز همان دور از بین می‌رود و نوبت به بازیکن بعدی منتقل می‌شود. بازیکن می‌تواند در هر لحظه با انتخاب گزینه Hold امتیاز فعلی خود را به امتیاز کل اضافه کند. اولین بازیکنی که به ۱۰۰ امتیاز برسد، برنده بازی خواهد بود. این پروژه برای تمرین مفاهیمی مانند دستکاری DOM، مدیریت رویدادها، منطق بازی، مدیریت وضعیت (State Management) و به‌روزرسانی پویا رابط کاربری طراحی شده .است

Pig Game – JavaScript Dice Game

Pig Game is a simple two-player dice game built with HTML, CSS, and JavaScript. Players take turns rolling a die to accumulate points. Each roll adds to the current score unless a 1 is rolled, which resets the current score and passes the turn to the other player. Players can choose to Hold at any time to add their current score to their total score. The first player to reach 100 points wins the game. This project demonstrates JavaScript fundamentals such as DOM manipulation, event handling, game logic, state management, and dynamic UI updates.
